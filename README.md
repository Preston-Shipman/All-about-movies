# All-about-movies

A app that utilizes New York Times and OMDb apis to search for movies.
Allows users to input a movie title and find news and reviews regarding the movie,

# Built with
- jQuery
- Bootstrap
- CSS
- HTML
- JavaScript
- AJAX
- NYT API
- OMDb API
